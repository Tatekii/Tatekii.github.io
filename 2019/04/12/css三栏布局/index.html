实现三栏布局的五种方法

本文最后更新于：2019年8月15日 下午

中间自适应，两边300px，高度已知

<artical class="parent">
    <div class="left"></div>
    <div class="center">
      <h1>center</h1>
    </div>
    <div class="right"></div>
</artical>

float

  • 左右div分别两边浮动
  • 默认div按照left，center，right排列，此处为例外，由于浮动元素不占据文档流，center要写right后面否则right挤到下一行
.left{
  float:left;
  background-color:red;  
  width:300px;
}

.right{
  float:right;
   background-color:yellow;
   width:300px;
}

.center{
   background-color:blue;
}

圣杯布局

  • 三栏都浮动
  • 容器创建BFC
  • 负margin将三栏移动到一行
  • 容器设置左右padding确保中间内容不会被挡住
  • 左右relative设置left/right值移动去padding出的空位
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="X-UA-Compatible" content="ie=edge">
  <title>Document</title>
  <style>
    * {
      padding: 0;
      margin: 0;
    }
    .container{
      padding: 0 200px;
      height: 200px;
    }
    .center {
      float: left;
      height: 200px;
      width: 100%;
      background: red;
    }

    .left {
      float: left;
      height: 200px;
      width: 200px;
      background: blue;
      position: relative;
      left: -100%;
      margin-left: -200px;
    }

    .right {
      float: left;
      height: 200px;
      width: 200px;
      background: yellow;
      position: relative;
      right: -200px;
      margin-left: -200px;
    }
  </style>
</head>

<body>
  <div class="container">
    <div class="center">center</div>
    <div class="left">left</div>
    <div class="right">right</div>
  </div>
</body>

</html>

双飞翼布局

  • 三栏都浮动
  • 容器创建BFC
  • 负margin将三栏移动到一行
  • 在center中加入一个子块，为子块设置margin避免被遮挡
  • center清除浮动
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="X-UA-Compatible" content="ie=edge">
  <title>Document</title>
  <style>
    * {
      padding: 0;
      margin: 0;
    }

    .container {
      float: left;
      height: 200px;
      width:100%;
    }

    .center {
      height: 200px;
      width: 100%;
      background: red;
      margin-left: 200px;
      margin-right: 200px;
    }

    .left {
      float: left;
      height: 200px;
      width: 200px;
      background: blue;
      margin-left: -100%;
    }

    .right {
      float: left;
      height: 200px;
      width: 200px;
      background: yellow;
      margin-left: -200px;
    }
  </style>
</head>

<body>
  <div class="container">
    <div class="center">center</div>
  </div>
  <div class="left">left</div>
  <div class="right">right</div>

</body>

</html>

绝对定位

所有div设置绝对定位
左右div设置宽度和left, right=0
中间div的left和right值分别为左右div宽度

div{
  min-height:200px;
  position:absolute
}
.left{
  left:0;
  background-color:red;  
  width:300px;
}

.right{
  right:0;
   background-color:yellow;
   width:300px;
}

.center{
   background-color:blue;
  left:300px;
  right:300px;
}

flex （推荐）

设置center的flex=1
设置flex后子元素的float、clear 和 vertical-align会失效


.parent{
  display:flex;
}
.left {
  background: green;
  flex-basis:300px;
}
.center {––
  background: red;
  flex: 1;
}
.right {
  background: orange;
  flex-basis: 300px;
}

table

先把容器撑开，给不同格子设置宽度

div{
  display:table-cell;
}
.parent{
  display:table;
  height:200px;
  width:100%;
}
.left{
  background-color:red;  
  width:300px;
}

.right{
   background-color:yellow;
   width:300px;
}

.center{
   background-color:blue;

}

grid

一样把容器撑开，设置三列，分别给宽度
网格布局 grid-template-rows设置行，columns设置列

.parent{
  display:grid;
  width:100%;
  grid-template-columns:300px auto 300px;
  grid-template-rows:200px;
}
.left{
  background-color:red;  
}

.right{
   background-color:yellow;
}

.center{
   background-color:blue;
}

如果改成竖直方向

  • 注意height100%是没用的，使用vh高度，视口高度
  • 例如flex，要改变布局方向，flex-direction=column
