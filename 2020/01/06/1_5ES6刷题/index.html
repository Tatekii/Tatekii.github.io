  3. const/let

    • 都没有变量提升
    • 临时性死区(TDZ)没有赋值的声明呆的地方
    • 都会创建块级作用域
    • 不能重复声明
    • const不允许再赋值（指向内存地址的指针不会改变）
    • const声明立即赋值
    • Object.freeze()使得对象的内容不允许在改变

  4. 展开运算符

//[1,2,3]

    • 数组拼接

let arr2 = [7,8,9,0]
console.log([...arr1,...arr2])

  7. for…of循环

    • for…of 循环对象的value值，用于可迭代对象（不能用来遍历普通对象）
    • for…in 循环对象的key值,不推荐用于遍历数组（数组使用for循环即可） 
      let arr = ['a', 'b', 'c'];
for (let pair of arr) {
  console.log(pair);
}
// a
// b
// c
for (let pair of arr.entries()) {
  console.log(pair);
}
// [0, 'a']
// [1, 'b']
// [2, 'c']

for (let pair of arr.keys()) {
  console.log(pair);
}
// 0
// 1
// 2

    • 使用var关键字声明或初始化的变量，会将声明语句“提升”到当前作用域的顶部。赋值语句不提升

  14. weakMap

    1. 键只能是对象
    2. 对键的引用是弱引用
    3. 不可迭代不可枚举，只支持new、has、get、set 和delete。
          const wm = new WeakMap();
    let key = Obj
    wm.set(key, 1);
    key = null;
    // wm对key所引用的对象只有弱引用，在key（对象）的引用被清除后，Obj可以被垃圾回收机制回收
    let map = new Map();
    let key = Obj
    map.set(key, 1);
    map.delete(key);
    key = null;
    // 即使key对Obj的的引用被清除，map中仍保持对Obj的强引用，垃圾回收机制不能回收这个Obj
    // 还需要从map中删除这个key才能彻底一刀两断
    ```
