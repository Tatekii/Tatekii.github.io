组件是react一等公民

组件：复用，独立，组合

创建组件

函数组件（使用函数创建）

  • 大写首字母
  • 必须return
  • 包括箭头函数
    function Com1(){
  return (
    <div>我是Com1</div>
  )
}
ReactDOM.render(<Com1 />,document.getElementById('app'))

    babel转译后,function运行在严格模式下,严格模式的this指向null

    类组件

    - 大写首字母
- extends自`React.Component`
- 必须return
class MyCom2 extends React.Component{
  render(){
    return (
      <div>我是Com2</div>
    )
  }
}
